# fprime-nucleo_h723zg-freertos-reference
This project is an implementation of F` on FreeRTOS for the NUCLEO-H723ZG board. 

## System Requirements
- F Prime System Requirements listed [here](https://fprime.jpl.nasa.gov/latest/docs/getting-started/installing-fprime/#system-requirements)
- Minimum required **CMake version: 3.27.0**

## Prerequisites
1. Follow the [Hello World Tutorial](https://fprime.jpl.nasa.gov/latest/tutorials-hello-world/docs/hello-world/)
